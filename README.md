## 🚀特性
1. bevy作为渲染引擎，具备ECS的高性能，高度的可扩展性。
2. wgpu作为图形API，具备跨平台，现代化图形API
3. rust作为开发语言，具备安全性，工具链完备性，可直接编译至WASM，充分发挥web端的性能。
4. cesium作为参考应用，具备实用性，GIS图形的精确性。
